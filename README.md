# Flood Safety Guidelines

**Flood Safety Guidelines** is a simple, beginner‑friendly frontend project built using **HTML and CSS**. It provides clear, practical guidance on how to stay safe **before, during, and after a flood**. This project was created for a Devpost hackathon to help communities prepare for emergencies.

---

## 🚀 Project Overview

Floods are a common natural disaster, and many people don’t know how to prepare or react safely. This website organizes safety instructions into three main phases:

- **Before a Flood:** Emergency planning, preparing a 3‑day emergency kit, home preparation, and staying informed.  
- **During a Flood:** Critical safety warnings, evacuation steps, avoiding hazards, and staying connected.  
- **After a Flood:** Health precautions, damage assessment, documentation for insurance, and community support.

Additional tips are included to help families stay prepared, such as keeping cash, laminated emergency contacts, and checklists.
